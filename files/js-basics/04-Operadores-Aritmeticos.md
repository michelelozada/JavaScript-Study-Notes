**Operadores aritméticos**  

&nbsp;   

**1. Adição** 
```js

let a = 4;
let b = 2;

console.log(a + b)  // Retorna: 6
```

&nbsp;   

**2. Subtração**  
```js

console.log(a - b)  // 2
```

&nbsp;   

**3. Multiplicação**  
```js

console.log(a * b)  // 8
```

&nbsp;   

**4. Divisão**  
```js

console.log(a / b)  // 2
```

&nbsp;   

**5. Módulo (resto da divisão)**  
```js

console.log(a % b)  // 0 
```

&nbsp;   

**6. Exponencial**  
```js

console.log (a ** b) // 16
```

&nbsp;   

**7. Incremento**  
```js

a++;
console.log(a) // 5
```

&nbsp;   

**8. Decremento**  
```js

b--;
console.log(b) // 1
```
